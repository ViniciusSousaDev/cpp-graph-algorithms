# Graph Algorithms in C++

Implementation of basic graph algorithms using adjacency matrix.

## Features

- Create graph
- Add edges
- Display adjacency matrix
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Example

Example graph:

0 -- 1 -- 3
|
2 -- 4

BFS (starting at 0):
0 1 2 3 4

DFS (starting at 0):
0 1 3 2 4

## Technologies

- C++
- STL queue
